# GachaGo 🌍

> Pull your next adventure. The world is your gacha machine.

## What It Is
GachaGo is a travel experience platform where you pull a gacha machine for any destination on Earth and receive an AI-generated adventure suggestion. Complete the activity, upload photo proof, and earn location-based cosmetics for your travel avatar.

## Features
- Interactive world map with pre-pinned destinations
- 4 rarity tiers — Common, Rare, Ultra Rare, Legendary
- Claude AI generates personalized suggestions per pull
- Photo proof of visit → cosmetic gacha reward
- Customizable avatar and cosmetic inventory
- Travel log with verified activity history

## Stack
- Frontend — Vanilla JS, HTML/CSS, Leaflet.js
- AI — Anthropic Claude API
- Maps — OpenStreetMap / Nominatim
